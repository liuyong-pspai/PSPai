🐉 小龙人 XiaoLongRen v1.2.0
================================

你的AI数字分身 — 能聊天、能干活、能记住你的AI助手。

📦 安装说明

  Windows:
    1. 解压此压缩包到任意目录
    2. 双击 install.bat
    3. 浏览器自动打开 → 点右上角⚙️配置大模型 → 填API Key → 开始使用

  macOS / Linux:
    1. 解压此压缩包到任意目录
    2. 运行 ./install.sh（或直接双击 index.html）
    3. 浏览器打开 → 点右上角⚙️配置大模型 → 填API Key → 开始使用

🔑 需要准备什么

  一个模型API Key（推荐DeepSeek，注册即送免费额度）：
  https://platform.deepseek.com/api_keys

  支持的模型提供商：
  - DeepSeek（推荐，免费额度）
  - OpenAI
  - Moonshot月之暗面
  - 智谱GLM
  - 通义千问

💻 系统要求

  任意现代浏览器（Chrome/Edge推荐，支持语音输入）
  不需要安装Python或其他软件
