# 🐉 小龙人 XiaoLongRen

**中国人自己研发的AI智能体** — 基于平行时空PSPAI架构的数字生命体产品。

## 快速开始

### 方式一：桌面版（推荐）
1. 从 [GitHub Releases](https://github.com/liuyong-pspai/PSPai/releases) 下载对应平台的安装包
2. 解压后运行 `start.sh`（Linux/macOS）或 `start.bat`（Windows）
3. 浏览器访问 http://localhost:8088

### 方式二：PWA 手机版
直接浏览器打开 `frontend/pwa/html/mobile.html` 即可使用

### 方式三：Android APK
从 GitHub Releases 下载 PSP3.6 APK 安装

## 配置
1. 复制 `.env.example` 为 `.env`
2. 填入你的 API Key
3. 重启程序

## 功能
- 6 种角色切换（龙渊/螭狱/清墨/霜华/夜莺/翎）
- 8 层记忆系统
- 40+ 工具调用
- 蜂群天犬智能搜索
- 自进化学习系统

## 技术架构
- 前端：纯 HTML/CSS/JS（PWA 兼容）
- 引擎：PSPAI（平行时空AI架构）
- 通信：WebSocket + HTTP
