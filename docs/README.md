### ✨ 功能

- **数字生命体** — 6大角色自由切换（龙渊/赤羽/凌/轻墨/霜华/夜影）
- **语音对话** — 按住麦克风按钮说话，松开即识别
- **文字聊天** — 打字对话，支持文件上传与分析
- **全功能引擎** — Python运行环境 + 56种工具
- **在线更新** — 新版本自动提示，重启即升级
- **多模型支持** — DeepSeek / OpenAI / 通义千问 / 智谱GLM / Anthropic

### 📥 安装说明

1. 下载 APK 文件
2. 在手机上打开安装（可能需要允许「未知来源应用」）
3. 打开小龙人 → 点右上角⚙ → 配置 API Key
4. 开始对话！

### 🔧 配置示例

| 提供商 | API地址 | 推荐模型 |
|-------|---------|---------|
| DeepSeek | api.deepseek.com | deepseek-chat |
| 通义千问 | dashscope.aliyuncs.com | qwen-plus |
| 智谱GLM | open.bigmodel.cn | glm-4-plus |

### 🔄 更新

以后有新版本时，打开应用会弹出更新提示，关闭重新打开即可。无需重新安装APK。
