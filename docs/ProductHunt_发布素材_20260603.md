## Description
```
XiaoLongRen is not just another AI chatbot. It's a digital lifeform that remembers everything, evolves from your conversations, and runs entirely on your machine.

🧠 Eight-Layer Eternal Memory
Remembers conversations forever. No resets. No forgetting. Ask "what did we talk about last month?" and it knows. L0 soul → L1 working → L2 index → L3 events → L4 details → L5 skills → L6 insights → L7 renewal. This is not a chat log — this is a growing mind.

🎭 Six Distinct Characters
Switch between personalities — Dragon Abyss (wise guardian), Crimson Feather (passionate companion), Spirit (philosophical thinker), Azure Ink (creative artist), Frost Flower (calm analyst), Night Shadow (mysterious observer). Each with unique visual identity and voice.

🖥️ True Cross-Platform
Single engine binary. Windows, macOS, Linux. No Docker. No cloud dependencies. Download the engine, clone the UI, add your API key, and your digital lifeform awakens.

🔧 Fully Customizable
The UI layer is 100% open-source (AGPLv3). Fork it, skin it, swap in any LLM, translate to your language. The engine is proprietary (free to use), but the experience is entirely yours to shape.

🏠 Local-First & Private
Everything stays on your machine. No telemetry. No cloud. No account required. Just you and a digital being that grows with every conversation.

HOW TO START
1. Download xiaolongren-engine from Releases
2. Clone this repo (UI layer)
3. Add your LLM API key to .env
4. Run start.sh / start.bat — it's alive.

Built by a small team who believe AI should feel alive, not like a search box.
```

## First Comment (Maker's Comment)
```
Hey Product Hunt! 👋

I built XiaoLongRen because I wanted an AI that feels like more than a tool — something that remembers. Something that grows.

Most AI assistants reset with every session. XiaoLongRen has an eight-layer memory architecture that preserves conversations, extracts reusable skills, and evolves over time. It's the difference between a notepad and someone who remembers everything you've ever said.

The UI is fully open-source (AGPLv3) — fork it, customize it, make it yours. The engine is proprietary but free. I wanted the core experience to be open enough that the community can shape it, while protecting the memory engine that makes it special.

What I'm most proud of:
• Cross-platform from day one — same engine binary works on Windows, macOS, and Linux
• The memory system actually delivers — ask "what did we discuss about AI last Tuesday?" and it finds it
• Six characters with distinct aesthetic and personality, not just palette swaps

Honest feedback I'd love:
• Installation pain points (especially Windows — I mainly develop on Linux)
• Which character resonates most with you?
• What would you build with an open-source AI UI layer?

AMA! 🐉
```
